What is Cody?

Cody is Sourcegraph's AI coding assistant that leverages their powerful code intelligence platform. Unlike other AI assistants, Cody can search and understand your entire codebase—including private repositories—making it ideal for large organizations.

Key Features

  • ▸ Codebase-Wide Search — Find anything across all your repos
  • ▸ Private Code Understanding — Can read your private repositories
  • ▸ Enterprise Security — SOC2 compliant, data stays in your org
  • ▸ VS Code & IntelliJ — Works in major IDEs
